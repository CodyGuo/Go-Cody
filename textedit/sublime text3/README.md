## 2. 常见问题
### 2.1 Tag安装后不能使用Ctrl + Alt + F 进行格式化
* 解决方法来自: http://www.java123.net/v/934921.html
  解决方法：在Github上下载压缩包版手动安装https://github.com/SublimeText/Tag 将解压的Tag-st3放在   sublime text目录\Data\Packages下，重启 
快捷键能用。
